Spread operator

			const size = {width: 10, height: 5};
			const extraSize = {depth: 3};
			const box = {...size, ...extraSize};
			// box === {width: 10, height: 5, depth: 3};
			const specialBox = {depth: 4, ...size, width: 8};

Rest operator

			// const box = {width: 10, height: 5, depth: 3};
			const {depth, ...size} = box;
			// depth === 3
			// size === {width: 10, height: 5}

Template literals

			const Box = ({width, height, depth, ...rest}) => {
			  const volume = width * height * depth;
			  return (
			    <div {...rest}>{`Volume: ${volume}`}</div>
			  );
			}

React Native

Основные компоненты

  • View — базовый блок;
  • Text — текстовый блок;
  • Image — картинка;
  • ScrollView, FlatList, SectionList — списки;
  • много других.

Нет CSS-классов, есть только стили

			<View style={
			  {width: 100, height: 50}
		    }/>
			<View style={
			  {width: 100, height: 50, backgroundColor: "skyblue"}
			}/>

Нет CSS-классов, есть только стили

			const styles = StyleSheet.create({
			  box: {width: 100, height: 100},
			  coloredBox: {backgroundColor: "skyblue"}
			});
		
			<View style={styles.box}/>
			<View style={[styles.box, styles.coloredBox]}/>

Композиция функций

			const sum = a => b => a + b;
			const mul = a => b => a * b;
			// x * 2 + 3
			const formula = compose(sum(3), mul(2));
			formula(5) === 13
